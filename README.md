# Pizza Shop Architecture Demo

This is a public demo repo to experiment with a vertical slice architecture with Orleans for a pizza shop application.

Core business domains (high level):
- Menu & Catalog
- Ordering
- Customer
- Delivery / Fulfillment
- Kitchen / Operations
- Payments
- Loyalty & Promotions
- Content & Marketing
- Store Management
- Analytics & Reporting

You can extend this repo with folders, services, and modules for each domain as you explore different architecture patterns.
